`Documentation`

"This is an API dedicated to compiling and organizing different queer films."

Request URL Method Parameters Request Format Response Format Description
Get All Movies /movies/ GET N/A N/A JSON containing a movie array Returns a list of all movies
Find Movie By Title /movies/:Title GET Title title (URL) JSON object with a single movie Returns the data of a specified movie
Get Movie By Genre /movies/genre/:genreName GET Genre genre (URL) JSON object with genre name Returns a movie's genre
Get Movies By Director /movies/director/:directorName GET Director director (URL) JSON object with data about director & their movies Returns a director's bio and work
Create a New User /users POST username (URL) N/A Returns a JSON object with user information Allows the creation of a new user account
Update a Username /users/:username PUT Username JSON object with update information Confirmation message Allows a user to update their profile
Add Movie to Favorites /users/:Username/FavoriteMovies/:movieID POST username, movieID N/A Confirmation message Allows users to favorite a movie
Remove a Movie from Favorites /users/:Username/FavoriteMovies/:movieID DELETE username, movieID (URL) N/A Confirmation message Allows a user to remove a favorite movie
Delete a profile /users/:Username DELETE username (URL) N/A Confirmation message Permanently deletes a user's profile
